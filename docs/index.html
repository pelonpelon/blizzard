So, what's inside?

Gumby Framework was built to be the most maleable grid framework you’ve ever used while still maintaining the familiarity and ease-of-use of the grid you’ve always used.

Grids

Flexible, portable, and rock-solid. The only grid system to go hybrid.

Forms

The most tedious user interface elements to style, styled for you.

Buttons

"The Tree of Clicking" has been planted for you. Copy. Paste. Convert.

Toggles & Switches

Calling these amazing is an understatement. JPL could use them to get us to Mars.

Sass

Built with the power of Sass & Compass. Experience the power of mixins, variables, nested rules and more...

Navigation

Gumby strikes a balance between quick access and an unobtrusive navigation on small screens.

Labels

Bring a user's focus and your applications goals in line with each other using quick, sexy labels.

Entypo Icons

Icons baked in! Responsive, resolution independent icons for your site.

Dropdowns

How awesome would it be if you didnt need .js for dropdowns? This awesome.

Tabs

You know what used to suck about tabs? Oh... yeah, we fixed that.

Drawers

Lets face it, these things are the coolest. Just like modals, but actually good.

Templates

PSD’s included; HTML too! Gumby Framework doesnt hate your design team.

Quick Start

  • Clone the Gumby repository from GitHub.
  • Open up settings.scss and tweak to your heart's content.
  • Compile the Sass and view the awesome changes in your browser.

Using the Customizer

  • Download a customized build with the required components, colors and fonts.
  • Open up settings.scss, check out the settings the customizer has modified and tweak to your heart's content.
  • Compile the Sass and view the awesome changes in your browser.

Browser Support

We support all major browsers with graceful degredation as far back as IE8.
